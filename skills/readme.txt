skills folder
=============

Purpose: main Skill prompt library for agent nodes — audit rules, response shape, checklists. The selected file is used in full as the system prompt.

Important: all working prompt files in this folder must be Markdown only — use the .md extension (e.g. generic-security-rules.md).

This readme.txt is an intro to the folder; define each skill in a separate .md file.
