lead_skills folder
==================

Purpose: alternative “lead” prompt library (Lead skill) for agent nodes in VulnFlow. The selected file is used as the system prompt; in the UI, Lead skill and Skill are mutually exclusive.

Important: all working prompt files in this folder must be Markdown only — use the .md extension (e.g. my-lead-rules.md).

This readme.txt is a short folder description; write agent instructions in separate .md files.
